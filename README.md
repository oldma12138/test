## ✨ 功能
1. **在线 / 离线** 均可使用
	- 访问我们通过 GitHub Pages 强力驱动的在线游戏平台进行云游戏。
	- 或者在 Releases 下载我们的最新版本全部源码，解压后打开 `index.html` 网页即可离线畅玩 。
2. 多种类型游戏供选择
	- 我们立志收集所有的开源可离线 h5 小游戏到这里来，不断更新迭代，添加更多玩法
